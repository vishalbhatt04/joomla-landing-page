Single Sign-On

miniOrange offers a single sign-on solution for Joomla. Log in to your Joomla site using Identity Providers that are SAML 2.0 compatible. Easily Configure the Identity Provider with the Joomla site by providing only the SAML login URL, IDP Entity ID, and Certificate.

The Joomla SAML 2.0 IDP enables users on your Joomla site to connect in to your SAML 2.0 compliant Service Provider. We provide assistance for all recognized Service Providers. The Joomla as IdP SAML SSO Plugin functions as a SAML 2.0 Identity Provider that can be set to build trust between the plugin and other SAML 2.0 compatible Service Providers.

The OAuth Client Single Sign-On plugin allows users from OAuth 2.0 competent OAuth/OpenID Providers to connect in to your Joomla site. OAuth / OpenID Connect Client plugin works with any OAuth/OpenID provider that adheres to the OAuth 2.0 or OpenID Connect 1.0 standard.

Joomla OAuth / OpenID Connect Server (OAuth 2.0 Server) enables Single Sign On to your Joomla client apps. It enables you to utilize Joomla as an OAuth / Openid Connect Server and access OAuth APIs. The Joomla OAuth Server plugin was designed in Joomla so that users may work with OAuth2 compliant clients.

Integration of Active Directory (AD) with Lightweight Directory Access Protocol (LDAP) for intranet sites. The plugin allows you to log in to your Joomla site using credentials saved on your LDAP server. It enables users to authenticate via NTLM and Kerberos.

miniOrange provides a ready to use solution for Joomla User Provisioning using SCIM (System for Cross-domain Identity Management) standard. This solution ensures that you can sync add, update, delete, and deactivate user operations with Joomla using the SCIM User Provisioner plugin.

Social Login, Social Sharing plugin is a Joomla plugin that allows your visitors to comment, share, login and register with many Social Media applications like for example Facebook, Twitter, Vkontakte, Google, LinkedIn, WindowsLive, Instagram, Salesforce and Amazon.

The Firebase Authentication plugin enables users to log in to their Joomla site using their Firebase credentials. You can easily use our Firebase Authentication plugin to allow users to login to your Joomla site using their Firebase credentials.

Two Factor Authentication (TFA) for your Joomla site that is both safe and simple to set up. MiniOrange Two Factor Authentication (TFA) offers a second layer of protection to your Joomla accounts, rather than depending just on a password, which may be phished or guessed. The Joomla OTP Verification plugin validates users' email and mobile phone numbers by issuing a verification code (OTP) after registration. It eliminates the possibility of a user enrolling on the Joomla site using a fake email address or mobile number. This plugin verifies the existence of the Email Address/Mobile Number as well as the user's ability to access that Email Address/Mobile Number.

The Joomla REST API authentication plugin prevents unwanted access to your Joomla site by verifying various authentication credentials

Login security, registration security, brute force attack protection, IP monitoring and blacklisting, DOS attack protection, and strong password enforcement are all offered by the network security plugin.
